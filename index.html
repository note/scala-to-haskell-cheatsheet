Defining functions
Scala

def double(x: Int): Int = 2 * x
Haskell

double :: Int -> Int
double x = 2 * x
Haskell (GHCi)

double :: Int -> Int; double x = 2 *x
Many tutorials use let double .... There's no difference between those two notations in GHCi so it's up to you to decide .
Haskell (GHCi) - alternative

:{
double :: Int -> Int
double x = 2 * x
:}
Note the :{ and :} in the first and last line. Besides of that it's the same as one would define the function in a source file.
